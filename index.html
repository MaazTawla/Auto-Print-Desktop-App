PDF scaling & paper

Order PDFs are printed via SumatraPDF (pdf-to-printer). Use Fit to page when the server sends a fixed page size and you want it scaled to the printer’s logical page. Optional paper size must match a name the driver accepts.

Examples: A4, Letter, 80mm — depends on the driver.
